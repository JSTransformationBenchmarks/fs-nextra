FS-Nextra

Node.js V8 native fs, enhanced with util.promisify and standard extra methods. Written in full ES2017, sans every sync method. Async is the future!

Docs

All fs methods which return a callback, return a promise instead. All fs-extra methods return a promise exclusively. No sync methods are included from either fs or fs-extra. Requires minimum Node.js v8.1.0.

The full documentation can be found here: fs-nextra.js.org.
